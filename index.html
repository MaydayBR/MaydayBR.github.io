My Work

News Article Scraping Project

I delved into web scraping using Python's requests and BeautifulSoup libraries, allowing me to extract meaningful content from HTML documents. I explored API integration, particularly with OpenAI's GPT models, which taught me how to harness powerful language models to generate insightful responses. Implementing Flask as a web framework enabled me to create a robust backend for handling requests and serving web pages. I set up environment variable management with dotenv, securing sensitive information such as API keys. Throughout the process, I implemented logging to monitor application behavior and debug effectively. I got hands-on experience with Retrieval-Augmented Generation (RAG), which involves retrieving relevant documents and generating responses based on that information, thereby improving the quality and relevance of generated content. This technique, often implemented using tools like LangChain, significantly enhances the application's ability to provide accurate and contextually appropriate answers.

YouTube Video Summary and Q&A App

After completing this new project, I gained valuable insights and hands-on experience with several new tools and techniques. I learned how to leverage the YouTube Transcript API to fetch video transcripts, which involved understanding how to extract and manipulate video IDs from URLs. This project also enhanced my skills in integrating OpenAI's GPT-3.5-turbo model to summarize transcripts and generate responses based on user questions. I improved my Flask application development skills, particularly in managing routes and handling JSON data with the request and jsonify modules. Implementing CORS (Cross-Origin Resource Sharing) with Flask-CORS was another critical aspect, ensuring my application could interact with resources from different origins. Through this project, I solidified my understanding of web scraping, API integration, and secure application development practices, which collectively broadened my technical expertise.
